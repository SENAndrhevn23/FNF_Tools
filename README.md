## Requirements

Before using this tool, make sure you have the following:

- **Windows 10 or higher**

## Setup

1. Open the Python script.  
2. Change the `ROOT_FOLDER` path to your own folder location. For example:

```python
# Original
ROOT_FOLDER = r"C:\Users\Example\Documents\Merged Tool"

# Change to
